# dotfiles
This is my dot files.
I use [stow](https://www.gnu.org/software/stow/) to manage my dotfiles
```
git clone https://github.com/jackiebin/dotfiles.git ~/dotfiles
cd ~/dotfiles
stow zsh # ...and whatever else you want
```
